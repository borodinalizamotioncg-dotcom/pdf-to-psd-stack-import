# PDF-to-PSD Stack Import

Import all pages of a PDF into one PSD document.
Each PDF page becomes a separate layer.

## Features

- Supports RGB, CMYK, Grayscale
- Custom DPI
- Optional reverse page order
- Progress bar (can be disabled)
- Silent mode for automation
- Automatically adjusts document size to the largest page

## Installation

1. Download the `.jsx` file
2. Copy it to:

   Windows:
   C:\Program Files\Adobe\Adobe Photoshop <version>\Presets\Scripts\

   Mac:
   Applications/Adobe Photoshop <version>/Presets/Scripts/

3. Restart Photoshop
4. Run:
   File ‚Üí Scripts ‚Üí PDF-to-PSD Stack Import

---

## Usage

1. Select a PDF
2. Choose DPI and color mode
3. Choose page order
4. Enable or disable Silent Mode
5. Click OK
